# 📚 Campus Sakshar – Frontend Project
# 🚀 Overview

Campus Sakshar is a student-focused web platform designed to simplify note-taking and peer-to-peer knowledge sharing within a campus environment.

This frontend project delivers a clean and responsive interface where students can explore, upload, and exchange academic notes efficiently. It focuses on usability, accessibility, and a smooth user experience.

#  🌟 Key Features

> 📝 Notes Upload Interface – Easy UI to add and manage notes

> 🔄 Peer-to-Peer Exchange – Share and access notes from other students

> 🧭 Navigation System – Structured pages for better usability

> 🎨 Modern UI Design – Clean layout with consistent styling

> 📱 Responsive Design – Works across devices (mobile, tablet, desktop)

> 🖥️ Screens / Pages Included

> 🏠 Home Page

> 🔐 Login / Signup Page 

> 📚 Notes Dashboard

> 📩 Contact / About Page

# 🛠️ Tech Stack

1. HTML5 – Structure

2. CSS3 – Styling (Flexbox, layout, responsiveness)

3. JavaScript (Vanilla JS) – Interactivity & DOM manipulation
